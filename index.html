Introduction

An open-source UI component library for building high-quality, accessible design systems and web apps using Vue.js.

Rebrand: Xperience Design UI

Presenting Xperience Design UI, the new identity of Radix Vue in its v2 evolution.

Xperience Design UI strives to deliver a low-level UI component library centered on accessibility, customization, and developer experience. Use these components as the foundation of your design system or integrate them progressively.

Our Principles

Accessibility-First

Accessibility is at the heart of Xperience Design UI. Our components align with WAI-ARIA design patterns to ensure that all users, regardless of abilities, can interact with your UI effectively. We handle intricate accessibility details like aria attributes, keyboard navigation, and focus management to simplify the developer's work.

Customizable & Unstyled

Xperience Design UI components come unstyled, providing developers the freedom to style them however they choose, using any CSS solution (vanilla CSS, preprocessors, or CSS-in-JS libraries). Our open component architecture allows you to wrap, extend, or modify each component as needed.

Open & Modular

Our components are designed to be open and adaptable, allowing you to customize each element to fit your needs. Whether adding event listeners, props, or refs, Xperience Design UI provides granular access to each component's inner workings.
